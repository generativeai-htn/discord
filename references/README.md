# สคริปต์สร้างห้อง Discord อัตโนมัติ

สคริปต์นี้จะสร้างหมวดหมู่และห้องต่างๆ ในเซิร์ฟเวอร์ Discord ของคุณ
ตามโครงสร้างที่ออกแบบไว้สำหรับโรงเรียน/วิทยาลัย (ดูรายละเอียดในไฟล์ setup-server.js)

## ขั้นตอนการใช้งาน

### 1. สร้างบอท Discord
1. ไปที่ https://discord.com/developers/applications
2. กด **New Application** ตั้งชื่อบอท
3. ไปที่แท็บ **Bot** → กด **Reset Token** เพื่อคัดลอกโทเคนบอท (เก็บเป็นความลับ ห้ามแชร์ให้ใคร)
4. เปิดสิทธิ์ **Privileged Gateway Intents** ตามต้องการ (ไม่จำเป็นสำหรับสคริปต์นี้)

### 2. เชิญบอทเข้าเซิร์ฟเวอร์
1. ไปที่แท็บ **OAuth2 > URL Generator**
2. เลือก scope: `bot`
3. เลือก permission: `Manage Channels`
4. คัดลอกลิงก์ที่ได้ แล้วเปิดในเบราว์เซอร์เพื่อเชิญบอทเข้าเซิร์ฟเวอร์ของคุณ

### 3. ตั้งค่าไฟล์ .env
สร้างไฟล์ชื่อ `.env` ในโฟลเดอร์เดียวกับ setup-server.js แล้วใส่:

```
BOT_TOKEN=โทเคนบอทของคุณ
GUILD_ID=ไอดีเซิร์ฟเวอร์ของคุณ
```

วิธีหา GUILD_ID: เปิด Discord → Settings → Advanced → เปิด Developer Mode
แล้วคลิกขวาที่ชื่อเซิร์ฟเวอร์ → Copy Server ID

### 4. ติดตั้งและรันสคริปต์

```bash
npm install
node setup-server.js
```

สคริปต์จะสร้างหมวดหมู่และห้องทั้งหมดให้อัตโนมัติ และจะข้ามห้อง/หมวดหมู่ที่มีชื่อซ้ำอยู่แล้ว
จึงสามารถรันซ้ำได้อย่างปลอดภัยหากต้องการเพิ่มห้องภายหลัง (แก้ไขในตัวแปร SERVER_STRUCTURE)

## หมายเหตุ
- ห้ามแชร์ BOT_TOKEN ให้ผู้อื่นหรือใส่ในโค้ดที่แชร์สาธารณะ
- สามารถแก้ไขชื่อหมวดหมู่/ห้อง หรือเพิ่ม-ลดห้องได้ในตัวแปร `SERVER_STRUCTURE` ในไฟล์ setup-server.js
